# Problem Statement

Managing personal finances can be challenging without proper tracking. Many people struggle to remember their income and expenses, leading to overspending or financial mismanagement. This project aims to provide a simple tool to log, monitor, and analyze personal transactions easily.

# Scope of the Project

The Personal Expense Tracker focuses on basic financial tracking for individuals. Users can record income and expenses, categorize them, and view summaries. The project ensures data persistence and easy retrieval of transaction history. While it is a console-based application, it can serve as a foundation for more advanced finance apps in the future.

# Target Users

- Students who want to track pocket money and expenses.
- Young professionals managing their personal finances.
- Anyone looking for a simple, lightweight tool to monitor income and expenses without complicated setups.

# High-Level Features

- Add and categorize income and expense transactions.
- View all recorded transactions in chronological order.
- Generate a summary showing total income, expenses, and balance.
- Analyze spending by category.
- Data storage using JSON for persistence across sessions.
